# AayatOS:

## Project Information
AayatOS is my browser-based operating system that I built using HTML, CSS and JavaScript. Through a clean and interactive interface, it presents my interests, aspirations and creativity. This project helped me learn front-end web development.

## What the Project Does
It welcoms visitors with a modern homepage and allows them to explore different sections-- a quote generator, info about the project and creator. 

## Features:

-OS inspired homepage with emojis
-Elegant pastel colour scheme
-Live digital clock
-Interactive quote generator with multiple inspirational quotes
-"Enter Desktop" button that smoothly scrolls to the Quote Generator
-About the Project section
-About the Creator section
-Soft titLe glow and subtle interface animations
-Hover animations for button and cards
-Smooth scrolling

## AI Usage
I used AI to help me troubleshoot a deployment issue and explain to me some CSS and JavaScript concepts as I wrote my code.
